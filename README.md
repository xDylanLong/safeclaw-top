## SafeClaw 是什么？

SafeClaw 不是普通 AI 聊天工具，而是一套按岗位拆分的 AI 工作团队。它帮助一人公司把重复发生的业务交给 AI 员工持续推进，例如自动获客、客户画像、社媒线索采集、自动私信、内容生产、视频转化、合同审查和业务复盘。
